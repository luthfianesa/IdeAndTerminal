Ide dan Terminal


CMD

dir -> Melihat list direktori
copy con namafilenya -> Membuat file
ctrl + z lalu enter -> Menyelesaikan proses pembuatan file
del namafilenya -> Menghapus file


Bash

ls -> Melihat daftar folder / direktori
cd -> Mengganti folder / direktori
mkdir -> Membuat direktori / folder
touch -> Membuat file
rm -> Menghapus file
rmdir -> Menghapus direktori
rm -rf -> Menghapus file atau direktori secara paksa
cd -> Mengganti direktori
cd namaDirektori -> Pindah satu langkah ke depan direktori yang dituju
cd namaDirektori/namaDirektori -> Pindah dua langkah ke depan direktori yang dituju
cd .. -> Mundur satu langkah ke belakang direktori yang dituju
cd ../.. -> Mundur dua langkah ke belakang direktori yang dituju

*Jika membuat file tanpa memberikan ekstensi di belakangnya maka secara default akan berekstensi .txt
